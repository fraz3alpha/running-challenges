What is this?

This browser extension helps you track your progress towards challenges and achievements which give you badges when completed:

  • View anyone's challenge progress by navigating to their 'All Results' page.
  • Minimise a challenge section by clicking on the mini-badge icon.

The 'Alphabeteer' challenge currently uses the English alphabet (A-Z) missing out X. In the future we aim to support selecting a different set of characters, or restricting this challenge (and others) by region. The letter is taken from the name as it is listed in the results summary tables as provided by parkrun.

Where does the data come from?

This extension mainly uses the data available on the existing parkrun webpage you are viewing, and then creates additional views of the data to display your challenge progress. The parkrun location information comes from the geo.xml file served by the parkrun website (which is used to draw the events page map). A processed version of this geographic data is cached in the extension to avoid having to repeatedly ask parkrun to provide this file - this keeps the plugin speedy, and does not put extra load on the parkrun website.

Where does the data go?

Nowhere. The data is entirely in your browser, nothing is sent or stored externally.

Options Page

The options page is available from the right-click menu of the extension icon in your toolbar, and also via the brower's extensions page.

In order for the extension icon to link directly to your stats it needs to know your parkrun athlete ID (so that it can generate the appropriate link). Add it here to make this work.

Translations

The translations page shows the current state of our attempt to support all the parkrun websites that are not in English. In order for us to correctly award volunteer badges for these sites we need to map the role name back in to English, and we have not yet found them all! Please have a look at the table, and if you know anyone who has done a missing role and has it listed on their athlete page in a language we haven't got yet, please drop us a line via email and let us know at TeamTCMakers@gmail.com .
